# [processon.com](https://processon.com/)

主要针对processon网站的个人文件主页，进行美化的样式设置：
- 隐藏页面顶部的一个广告图片
- 隐藏页面顶部的一个超链接按钮
- 增加了网页的背景颜色，看起来更清爽、漂亮。
